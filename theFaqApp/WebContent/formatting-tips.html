Text formatting tips for info sections

💡 Pro Tip: Keep it simple and stay happy!

1. You can use tags like <b>, <ol>, <ul>, <li>, <h4> etc to format textual data in info sections. Use of <br> tag is not needed. Do not use other tags that might break UI, which you can validate in home page.

2. To include a code snippet along with your info,

  1. make sure you do proper indentation in notepad or eclipse,
  2. trim additional whitespaces after step 1, and
  3. include the code inside code tags as follows:
    [code=java]//your code[/code]

    A good indented code looks like this:

    [code=java]
    class Test {
        public static void main(String[] args) {
            System.out.println("Hello World!");
        }
    }
    [/code]

Currently, only java, js, sql and xml are suported.
Multiple code snippets can be added however you should not nest them.

3. The module and submodule names are automatically title cased, left and right trimmed, and saved. Code snippets automatically get a linebreak before and after.

4. Avoid passing scripts or other syntactic sugars. The app is not designed to handle and data might become corrupt or permanently lost.
